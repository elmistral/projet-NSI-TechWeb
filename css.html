CSS

CSS est un langage qui permet de définir la taille, la couleur, la police, la disposition et d'autres aspects de l'apparence des éléments HTML.

Nous allons voir les bases du langage CSS :

1. Intégration d'une page CSS

< head >

< meta charset = "UTF-8" >

< title > Titre du site < / title >

< link rel = "icon" type = "image/x-icon" href = "images\logo-webtech.png" >

< link rel = "stylesheet" type = "text/css" href = "style.css" >

< / head >
/*Page CSS*/
...

Pour mettre du style à votre site web, vous devez implémenter sur chaque page la feuille de style, codée en CSS. Une seule feuille de style suffit pour le site entier et non par page web.

2. Sélecteur CSS

< p > Un paragraphe < / p >
< h1 > Un titre < / h1 >
p {

color : blue ;

}
 h1 {

color : green ;

}

Pour rajouter un style à une balise, nous appellons cette dernière, dans le css et nous indiquons le style que nous souhaitons obtenir.

3. Utilisation des class et des id

< h1 id = "titre" > Un titre < h1 / >

< p class = "texte" > Un paragraphe 1 < p / >
< p class = "texte" > Un paragraphe 2 < p / >
#titre {

color : #010d7b ;

}
 .texte {

color : blue ;

}

Pour rajouter un style à une ou plusieurs balises, nous leur déterminons une même class et nous l'appellons en CSS, avec un . devant son nom.
Cependant, quand vous souhaitez attribuer un style à un élément unique, vous prévilégiez plutôt un id, que nous appellons ensuite en CSS, avec un # devant son nom.

/*Choisir une police d'écriture*/
 .police {

font-family : 'Courier New' , Courier , monospace ;

}
/*Choisir la taille d'écriture en em, px, %, ...*/
 .taille {

font-size : 40 px ;

}
/*Changer l'épaisseur du texte lighter = fin, normal = normal et bold = gras*/
 .epaisseur {

font-weight : bold ;

}
/*Mettre un texte en italique*/
 .italique

font-style : italic ;

}
/*Décorer du texte underline = souligné et line-through = barré*/
 .decoration {

text-decoration : underline ;

}
/*Aligner un texte left = aligner à gauche, center = centrer un texte, right = aligner à droite et justify = justifier le texte*/
 .alignement {

text-align : center ;

}
/*Ajouter un alinéa*/
 .alinea {

text-indent : 20px ;

}
/*Mettre une couleur sur un texte*/
 .couleur {

color : pink ;

}
/*Ajouter une couleur de fond*/
 .fond {

background-color : yellow ;

}
/*Opacité d'un texte*/

Pour ajouter une marge extérieure à votre bloc, nous utiliserons le paramètre "margin", par contre, pour une marge intérieure à votre bloc, cela sera le paramètre "padding".
Dans chacun des paramètres, les valeurs représentent respectivement les marges haute, droite, basse et gauche.
