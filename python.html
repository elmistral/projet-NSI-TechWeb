Python est un langage de programmation facile à apprendre et à utiliser, ce qui le rend idéal pour une grande variété de projets.

Nous allons commencer par voir les bases de ce langage de programmation :

1. Afficher du texte

print ( "Hello World" )
Hello World

Le mot qui est entre guillemets va être écrit dans le terminal.

2. Entrée

n = input ( 'Entrer un nombre : ' )
Entrer un nombre : ...

La commande input va demander à l'utilisateur de répondre.

3. Variable

n = 1
print ( "Le nombre n est égal à " ,n)
Le nombre n est égal à 1

Nous pouvons attribuer une valeur à une lettre pour former une variable. Puis, nous l'affichons grâce à la commande print() vu précédemment.
