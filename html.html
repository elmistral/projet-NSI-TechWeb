HTML

HTML est un langage qui permet de créer des pages web. Il est utilisé pour structurer le contenu des pages web, comme les titres, les paragraphes, les images et les liens...

Nous allons voir les bases du langage HTML :

1. Écrire des titres

< h1 > Titre 1 < / h1 >
< h2 > Titre 2 < / h2 >
< h3 > Titre 3 < / h3 >

Titre 1

Titre 2

Titre 3

Plus le nombre du "h" est grand (1,2,3,4,5,6), plus le titre sera de moindre importance (sous-titres). Il faut respecter l'ordre hiérarchique des titres pour chaque page du site web. Cependant, il ne faut qu'un seul titre h1 par page web, ça sera en principe le titre unique de la page.

2. Écrire des paragraphes

< p > Paragraphe < / p >

Titre 1

La balise "p" permet d'écrire les paragraphes sur le site web.

3. Balises orphelines

< p > Paragraphe < / p > < / br >
< p > Paragraphe 2 < / p >

Paragraphe


Paragraphe 2

Il faut savoir que chaque balise HTML doit avoir une ouvrante (<...>) et une fermante (</...>). Mais, il existe des balises dîtes "orphelines" (elles sont apellées comme cela, car elles n'ont pas besoin d'une balise ouvrante et fermante contrairement aux autres balises HTML); par exemple la balise </br> permet un retour à la ligne forcé.

4. Mettre des liens

< a href = "https://www.sainte-anne-brest.fr/" > Sainte-Anne Brest < / a >
Sainte-Anne Brest

La balise <a> permet d'implémenter des liens cliquables, dans un site web.

5. Afficher des images

< img src = images\logo-webtech.png alt = Logo de Tech'Web / >

La balise <img> permet d'afficher une image. Avec le paramètre "src", nous allons chercher le lien vers l'image, et pour finir, le paramètre "alt" permet d'afficher un texte, si l'image ne fonctionne pas.

6. La stucture du code HTML

< ! DOCTYPE  html >
< html >

< head >

< meta charset = "UTF-8" >

< title > Titre du site < / title >

< link rel = "icon" type = "image/x-icon" href = "images\logo-webtech.png" >

< / head >

< body >

< p > Main ou corps du site < / p >

< / body >

< footer >

< p > Pied du site < / p >

< / footer >

< / html >
...

Les balises <DOCTYPE> et <html> permettent de définir que c'est du code HTML5.
La balise <head> comporte les paramètres du site : "meta" pour permettre d'écrire des caractères spéciaux tels que des é, à ou ù; "title" pour indiquer le titre du site dans l'onglet de navigation et "link" pour insérer le logo du site sur l'onglet de navigation.
La balise <body> permet d'écrire le corps du site.
Pour finir, la balise <footer> permet d'insérer un Pied du site.
